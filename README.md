# NFA to DFA

## Input

Each NFA must be saved as a .txt file individually inside '/NFAs' directory for the program to run properly.

## Output

The output will be saved as a .txt file inside '/DFAs' directory.

## How to run

```sh
python3 nfatodfa.py
```
